<h1>🍋 Lemon Empire 🐱</h1>
Lemon Empire es un simulador de gestión de negocios (Tycoon) desarrollado como proyecto final de Frontend. El objetivo del jugador es administrar un puesto de bebidas, gestionando inventarios, compras estratégicas y ventas diarias bajo una estética pixel art vibrante y minimalista. <br>

<h2>🌿 Concepto del Proyecto</h2>
Inspirado en los juegos clásicos de simulación, Lemon Empire combina la lógica de programación con una experiencia visual fluida. El proyecto se centra en el State Management (manejo de estados) de una economía simple: equilibrio entre gastos de insumos y margen de ganancia, todo envuelto en una narrativa amigable protagonizada por personajes animales.
<h2>🚀 Tecnologías Utilizadas</h2>
<ul>
  <li>Figma: Diseño de interfaz, prototipado de flujo y gestión de assets visuales.</li>
  <li>HTML5: Estructura semántica avanzada para separar las pantallas de juego, menús y diálogos.</li>
  <li>SASS & CSS3: Arquitectura de estilos escalable mediante variables, mixins para botones y anidamiento para componentes complejos como el "Mercado del Oso".</li>
  <li>JavaScript (ES6+): Motor lógico del juego, incluyendo manipulación del DOM en tiempo real y Local Storage para persistencia de datos.</li>
  <li>Bootstrap 5: Implementación de sistemas de grillas (Grid) y componentes responsivos como el carrusel de instrucciones.</li>
  <li>Toastify JS: Sistema de notificaciones elegante para el feedback del usuario (compras, errores de saldo, etc.).</li>
</ul>

<h2>📂 Estructura del Imperio</h2>
El sitio consta de 4 módulos principales:
<ol>
  <li>Dashboard de Inicio: Pantalla de bienvenida con acceso a manuales y configuración de audio.</li>
  <li>Centro de Gestión: El menú principal donde el jugador decide su estrategia (Ver inventario, comprar o vender).</li>
  <li>Mercado del Distribuidor: Interfaz de e-commerce interna para la adquisición de materia prima (limones, azúcar, vasos, agua).</li>
  <li>Simulador de Ventas: Pantalla de interacción con clientes y cierre de jornada económica con cálculo de ganancias.</li>
</ol>
